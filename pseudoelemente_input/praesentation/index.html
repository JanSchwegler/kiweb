Pseudoklassen

Mithilfe von Pseudoklassen kann ein Element nur dann verändert werden, wenn dieses bestimmte Vorgaben verfüllt. Pseudoklassen werden als erweiterung zu den normalen CSS Selektoren verwendet um so die Auswahl des Elementes zu spezifizieren. Häufig werden Pseudoklassen verwendet um beispielsweise jedes Zweite Element anzupassen oder das Element zu verändern sobald mensch mit dem Cursor darüber fährt. Das zweite beschriebene Szenario wird "hover" genannt. Nach dem normalen CSS Selektor wird mit einem ":" die Pseudoklasse angehängt. Das sieht beispielsweise wie folgt aus:

.button:hover {
 background-color: #0d6158;
}

Mit diesem Code wird allem Elementen mit der Klasse "button" eine Hintergrundfarbe vergeben, faslls ein User mit der Maus über das Element fährt.

Pseudoelemente

Grundsätzlich wird der Inhalt einer Website in HTML defineirt und nur diese Elemente (HTML-Tags) können mithilfe von CSS bearbeitet und gestaltet werden. Mit den sogenannten Pseudoelementen können jedoch Abschnitte bearbeitet werden, welche nicht den HTML Tags entsprechen. Aus meiner Sicht lassen sich Pseudoelemente in zwei Kategorien einteilen. Denn mit Pseudoelemten lassen sich auf der einen Seite nur einzelne Teile eines bestehenden Elementes auswählen und bearbeiten aber auf der anderen Seite können auf diese Weise auch per CSS eigene Elemente erzeugt werden welche nicht im HTML ersichtlich sind. Pseudoelemente werden im Vergleich zu Pseudoklassen mit zwei ":" angegeben. Mit diesem Wissen können Pseudoklassen und Pseudoelemente eifach voneinander differenziert werden.

Beispielsweise kann mit dem Pseudoelement "::fist-line" die Erste Zeile eines Textes ausgewählt und verändert werden:

p::first-letter {
 font-size: 5rem;
}

Mithilfe von "::before" & "::after" können Elemente in CSS erstellt und definiert werden. Diese können beide pro Element je nur 1 mal vergeben werden und werden vor bzw. nach dem eigentlichen Inhalt des Elements eingefügt. Um Before und After auf der Website sichtbar zu haben, müssen diese einen defineirten Inhalt haben. Sobald der Inhalt gelöscht wird, verschwindet das gesamte Pseudoelement.
